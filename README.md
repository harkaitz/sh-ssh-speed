# SSH-SPEED

Calculate the upload and download speed using SSH.

## Dependencies

- ssh (OpenSSH).
- dd  (GNU Coreutils).

## Help

ssh-speed

    Usage: ssh-speed [USER@]HOST
    
    Tell the upload and download speed over SSH.
